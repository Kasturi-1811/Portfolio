SkillSwap – Mutual Skill Exchange Platform

I developed a website called SkillSwap that helps people connect and exchange skills with one another. Users can post what skills they want to learn and what skills they can teach, making it a mutual learning platform.

This project was built during a one-day (9 AM – 5 PM) online hackathon. It promotes peer-to-peer learning by letting users search and connect with others who have complementary skills.

The backend was implemented using Django, and the platform allows account creation, skill listings, and user matching functionalities.

Technologies Used: Django, Python, HTML, CSS, JavaScript
Duration: 1-day Hackathon
